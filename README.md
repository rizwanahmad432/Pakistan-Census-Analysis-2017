# 📊 Pakistan Population Census 2017 Analysis

This repository contains an Exploratory Data Analysis (EDA) of Pakistan's Population Census 2017 dataset using Python.

## 📂 Files Included

- **Dataset:** Pakistan Population Census 2017  
- **Notebook:** `Rizwan_Census_Analysis.ipynb`  
- **README:** Project documentation  

## 📍 Dataset Source

Kaggle Dataset:  
https://www.kaggle.com/datasets/mabdullahsajid/population-of-pakistan-dataset

## 🛠️ Tools Used

- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## 📈 Analysis Includes

- Population distribution analysis  
- Urban vs Rural comparison  
- Gender ratio insights  
- Growth trend analysis
